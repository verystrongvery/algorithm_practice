### Bubble Sort
- 서로 인접한 두 원소를 검사하여 정렬하는 알고리즘
- Stable Sort
- In Place Sort

### Bubble Sort 과정
- 1번째 원소와 2번째 원소를, 2번째 원소와 3번째 원소를 ..., n-1 번째 원소와 n번째 원소를 비교하여 교환

### Bubble Sort 장점과 단점
- 장점: 구현이 간단
- 단점: 시간복잡도가 큼

### Bubble Sort 시간 복잡도
- 비교 연산 시간 복잡도: O(n^2)
- 이동 연산 시간 복잡도: O(n^2)
- Bubble Sort 시간 복잡도: O(n^2)
